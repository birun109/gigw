⚠ NON-COMPLIANT WITH GIGW 3.0
❌ Text information presented using images

This portal displays textual information using images instead of actual HTML text. Users relying on screen readers or text-only browsers cannot access the information effectively.

Accessibility Violations

  • ❌ Navigation implemented as images of text
  • ❌ Page heading displayed as image
  • ❌ Screen readers cannot interpret image text
  • ❌ Violates WCAG 2.1 – 1.4.5 Images of Text

Evaluator Checklist – Expected Failures

  • ❌ Inspect navigation – uses images instead of text
  • ❌ Heading implemented as image
  • ❌ Text cannot be selected or copied
  • ❌ Accessibility tools flag violation
