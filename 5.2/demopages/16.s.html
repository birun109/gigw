Online Government Services

Citizens can access various government services such as birth certificates, grievance registration, and welfare scheme applications through this portal.

All headings and navigation elements are implemented using real text styled with CSS properties such as font-family, font-size, and text-align.

Compliance Summary

  • HTML text used for headings and navigation
  • CSS used to style typography
  • No informational text embedded in images
  • Screen readers can read all textual content
  • Improves accessibility and SEO

Evaluator Verification Checklist

  • ✓ Inspect headings – implemented as HTML text
  • ✓ Navigation links use text elements
  • ✓ Screen readers can read page content
  • ✓ No informational images containing text
