❌ GIGW 3.0 Clause 5.2.21 – Keyboard Accessibility Failure
❌ Custom control is NOT keyboard accessible

Custom Div Button (FAIL)

Custom Div Button (FAIL)

Test Instructions:

  • Press Tab → Focus will skip this element.
  • Try Enter → Nothing happens.
  • Try Spacebar → Nothing happens.
  • Mouse click works → Keyboard does NOT.

Why This Fails Clause 5.2.21

  • ❌ Not reachable via keyboard
  • ❌ No tabindex="0"
  • ❌ No keydown handler for Enter/Space
  • ❌ Uses non-semantic element instead of <button>
  • ❌ Violates keyboard operability requirement

How to Fix (Concept Only)

  • Add tabindex="0"
  • Add role="button"
  • Add keydown handler for Enter and Space
  • OR simply use a native <button> element (recommended)
